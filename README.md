# Markdown cheatsheet
> Learning markdown step by step

- ### Markdown **目的**：
   - 用來作為一種網路內容的寫作用語言。
- ### Markdown **目標**：
   - 用來是實現「易讀易寫」。
- ### Markdown **語法**：
   - 全由標點符號所組成。
